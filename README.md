## Tradogate

Tradogate is a trading platform focused on helping users build a financial identity, understand market reality, and make better trading decisions.

## Description

Tradogate provides an opinionated, modern web interface for exploring trading features such as financial identity graphs, reality streams, trade rooms, and risk management tooling.  
This repository contains the frontend application built with React, TypeScript, Vite, Tailwind CSS, and ShadCN UI components.

## Setup Instructions

### Prerequisites

- Node.js (LTS or later) installed on your machine
- npm (comes with Node.js)

### Installation

1. Clone or download this repository.
2. Open a terminal in the project directory:

   ```bash
   cd sync-build-zone-main
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

### Running the Project Locally

Start the development server:

```bash
npm run dev
```

By default, Vite will start the app and print a local URL (for example `http://localhost:8080/`).  
Open that URL in your browser to view the Tradogate interface.

### Building for Production

To create an optimized production build:

```bash
npm run build
```

You can then preview the production build with:

```bash
npm run preview
```

## Features

- **Modern tech stack**: React, TypeScript, Vite, Tailwind CSS, and ShadCN UI.
- **Home experience**: Hero, feature highlights, testimonials, and calls-to-action tailored for trading workflows.
- **Pricing page**: Dedicated pricing view accessible from the main landing page.
- **Authentication pages**: Signup, login, and password reset screens wired to a Supabase backend (configuration via `.env`).
- **Responsive design**: Optimized layout for desktop and mobile screens.
